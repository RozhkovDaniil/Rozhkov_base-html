HTML-документ - это обычный текстовый документ. может быть создан как в обычном текстовом редакторе (Блокнот), так и в специализированном, с подсветкой кода(Notepad++, Visual Studio Code и т.п.). HTML-документ имеет расширение .html.

HTML-документ состоит из дерева HTML-элементов и текста. Каждый элемент обозначается в исходном документе начальнным (открывающим) и конечным (закрывающим) тегом (за редким исключением)

Начальный тег показывает, где начинается элемент, конечный - где закенчивается. Закрывающий тег образуется путем слеша / перед именем тега:<имя тега>...<имя тега>. Между начальным и закрывающим тегами находится некое содержимое элемента - контент.

Например:

  1. < р>Контент< /р> - создает параграф
  2. < b>Контент< /b> - Делает полужирное начертание
  3. < i>Контент< /i> - Курсивное начертание
